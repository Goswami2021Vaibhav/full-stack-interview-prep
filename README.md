# Interview Questions

A categorized collection of interview preparation notes, written in Q&A format. Each topic is broken down into focused categories (core concepts, functions, OOP, etc.) so notes are easy to navigate and extend over time.

## How notes are written

- Each category file is a standalone `.md` with a list of `### Q.` / `**A:**` entries.
- Answers are kept concise, with short code snippets where useful for quick revision.
- Every topic folder has its own `README.md` indexing its categories.
