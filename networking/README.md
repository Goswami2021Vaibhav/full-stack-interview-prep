# Networking

Interview Q&A notes for **Networking**, organized by category.

## How to Use These Notes

- 🟢 Easy · 🟡 Medium · 🔴 Hard — difficulty tag on every question.
- Questions are numbered sequentially within each file (Easy → Medium → Hard), matching the Table of Contents.
- Each category file has its own Table of Contents grouped by difficulty.
- **Follow-up questions** are listed without answers — interviewers often ask these next, so think them through yourself first.
- A **Real-life example** is included only when one genuinely helps, not on every question.
